## About

A Book Store eCommerce application made with NestJS and ReactJS.

## Tech used

* NodeJS
* NestJS
* MongoDB
* ReactJS
* Vite
* MaterialUI
* Redux
* Redux Toolkit
* Cookies
* JWTs
* Encryption
* Session Management

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
